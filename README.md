# AWS Cloud Web Hosting & Deployment Project

## Overview
This project demonstrates hosting a full web application on AWS EC2 (Amazon Linux). It includes:
- Web server (Apache)
- PHP backend for uploads
- Custom domain (Dynamic DNS / Freenom)
- HTTPS via Let’s Encrypt
- Admin dashboard to manage uploaded files

## Technologies Used
- **AWS EC2 (Amazon Linux 2)**
- **Apache HTTP Server**
- **PHP 8.x**
- **HTML/CSS/JS**
- **Let’s Encrypt SSL**
- **Cron jobs for automation**
